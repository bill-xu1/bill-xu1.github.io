My discord bot.

Bot usage:

It keeps a leaderboard of the highest used words in a server and the most active members.

Languages used:

This bot involved usage of Python and SQLite.

Specific libraries/frameworks:

  • Discord.py
  • SQLite3

How it works:

The bot uses discord.py to communicate with Discords API. It uses async and await methods in python. It reacts to events when a message is sent in a server and obtains the text of the message sent and the author's nickname. The bot then filters the message and adds it to an SQLite database that keeps track of the total words used in the server. As well as this it keeps track of who sends a message with the SQLite database and keeps track of the number of messages sent. When the data is requested, we take the SQLite data and outputs the highest used words and most active members
