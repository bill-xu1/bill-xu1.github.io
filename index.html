About the website

This website uses the popular CSS framework Bootstrap. I considered using a static website generator such as Jekyll/Hexo/Hugo, however bootstrap seemed the most customisable. It utilises the containers provided to create a DOM (website) that is easy to follow. It is a series of containers layered on top of each other. The portfolio site are certain containers bundled with a portion of an example used. There have been minor visual adjustments to the CSS to fit my preferences to provide a clean UI.
